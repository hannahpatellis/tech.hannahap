Software I use

  • WebOne HTTP Proxy Server: WebOne is a proxy server for devices that don't support modern SSL or the modern web in general. Built using Microsoft .NET 6.0, it will run on Windows or Linux systems. I run this using a Raspberry Pi Model 3B with Raspberry Pi OS Lite connected to a network via ethernet
      • Alternative web proxy packages
    • Web Rendering Proxy
    • Legacy Proxy
  • WaybackProxy: This is a slightly different proxy and tunnels requests through the Internet Archive's Wayback Machine
  • Disk Jockey: Simple-to-use disk image creator and analyser for your retro computers (or emulators!)
  • balenaEtcher: Flash OS images to SD cards & USB drives, safely and easily

Hardware I use

  • LILIWELL industrial CompactFlash card: I have found these work well in CF-to-IDE adapters as they are in fixed disk by default
  • TP-Link TL-SG105, 5 Port Gigabit Unmanaged Ethernet Switch
  • TP-Link AV1000, Powerline Ethernet Adapter Kit: I don't get close to 1Gbps with these, but they are reliable
  • BrosTrend WiFi to Ethernet Adapter
